Asignar Materias

Esta página te permite registrar a un estudiante y asignar materias de un programa de créditos.

Características del Proyecto

  • Desarrollado con Angular y .NET Core 7.
  • Implementa la arquitectura limpia siguiendo un template de Ardalis.
  • Utiliza Minimal Endpoints para la creación de endpoints de manera sencilla.
  • Emplea Guard Clauses para mejorar la legibilidad y mantenibilidad del código.
  • Se utiliza Implicit Operator para el mapeo de entidades, evitando el uso de librerías como AutoMapper.

Funcionalidades

El sistema cumple con los siguientes puntos:

  1. Permite a un usuario realizar un registro en línea de un estudiante.
  2. Los estudiantes se adhieren a un programa de créditos.
  3. Existen 10 materias en el programa, cada una equivalente a 3 créditos.
  4. Los estudiantes pueden seleccionar hasta 3 materias.
  5. Hay 5 profesores que dictan 2 materias cada uno.
  6. Se evita que un estudiante tenga clases con el mismo profesor.
  7. Los estudiantes pueden ver en línea los registros de otros compañeros.
  8. Los estudiantes solo ven el nombre de los alumnos con quienes compartirán cada clase.
